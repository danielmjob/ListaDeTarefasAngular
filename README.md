# ListaDeTarefasAngular

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.0.6.

## Development server

To start a local development server, run:
http://localhost:4200/

```bash
ng serve
```

## Json server

To start a local Json server, run:
http://localhost:3000/

```bash
json-server db.json
```
